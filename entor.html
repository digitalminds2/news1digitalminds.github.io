Componentes entorno de desarrollo

Lenguaje de marcado de hipertexto

HTML

Es el lenguaje de marcado estándar utilizado para crear y estructurar el contenido de páginas web

JavaScript

JavaScript es un lenguaje de programación de alto nivel que se utiliza en el desarrollo web para agregar interactividad y funcionalidad a las páginas.

CSS

se utiliza en una página web para definir la apariencia y el diseño de los elementos HTML. Es un lenguaje de estilo utilizado para controlar la presentación visual de una página web y separar el contenido del diseño.

Bootstrap

Bootstrap es un framework de código abierto muy popular que se utiliza en el desarrollo web para crear rápidamente sitios web y aplicaciones con un diseño moderno y responsivo. Proporciona un conjunto de herramientas, estilos predefinidos y componentes listos para usar, lo que facilita la creación de interfaces de usuario atractivas y consistentes.

Fuentes web de Google

Son una colección de tipografías gratuitas que puedes utilizar en tu página web. Estas fuentes son accesibles a través de un servicio alojado por Google, lo que significa que no necesitas descargar e instalar las fuentes en tu servidor web.
